Técnologias

  • JavaScript

    Linguagem de programação que torna as páginas web dinâmicas. Interage com elementos HTML, oferecendo funcionalidades interativas.

  • CSharp

    Linguagem de programação desenvolvida pela Microsoft. Usada para construir aplicativos Windows, web e móveis, integrando-se com a plataforma .NET.

  • Visual Basic

    Linguagem de programação da Microsoft. Versátil e fácil de aprender, usada para desenvolvimento de aplicativos Windows e web.

  • Angular

    Framework JavaScript mantido pelo Google. Facilita o desenvolvimento de aplicativos web robustos e escaláveis.

  • HTML

    Linguagem de marcação para estruturar páginas web. Define a estrutura básica, utilizando tags para elementos como texto, imagens e links.

  • CSS

    Linguagem de estilo que estiliza documentos HTML. Controla layout, cores e estética, garantindo uma apresentação visual atraente.

  • SQL Server

    Sistema de gerenciamento de banco de dados relacional da Microsoft. Armazena, recupera e gerencia dados de maneira eficiente.
